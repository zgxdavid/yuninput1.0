# Handoff / Release 文档同步清单

更新时间：2026-05-12

用途：每次 handoff 和每次 release 前，按本清单逐项确认，避免遗漏用户可见文档更新。

## 1. 必更文档

1. 对话记录.md
2. 真实对话.md
3. 匀码输入法说明书.md
4. YuninputManual.zh-CN.md

## 2. 必查一致性

1. 版本号一致：README、说明书、MSI 文件名、脚本参数中的版本号保持一致。
2. 更新日期一致：说明书与交接记录写明本轮更新时间。
3. 变更摘要一致：新增功能、行为变更、已知限制在中英文或主副文档中语义一致。

## 3. Handoff 必填

1. 在 diagnostics/handoff_*.md 明确写出：以上 4 份文档已更新。
2. 记录本轮新增功能和影响范围。
3. 记录是否完成构建、打包、安装验证；未完成项写清原因和接手建议。

## 4. Release 必填

1. 在发布说明中写清本轮核心变更。
2. 确认安装包版本与文档版本一致。
3. 若某步骤未完成，禁止标记“可发布”。
