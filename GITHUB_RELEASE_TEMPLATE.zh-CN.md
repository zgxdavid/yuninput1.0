## 仓库简介模板

匀码输入法（yuninput）是一个面向 Windows 11 TSF（Text Services Framework）的中文输入法项目。

当前仓库重点提供：

- 基于 C++ 的 TSF Text Service 骨架
- 可加载外部码表的输入引擎与候选窗能力
- 用户词频、置顶、屏蔽、上下文学习等基础机制
- 配置工具、安装脚本、EXE/MSI 安装包生成链路

本项目当前处于持续开发阶段，目标是实现一个可扩展、可维护、可验证的中文输入法框架，并逐步完善类似郑码输入体验的能力。
