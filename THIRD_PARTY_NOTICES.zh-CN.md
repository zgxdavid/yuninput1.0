# 第三方声明

本仓库包含由 yuninput 项目编写的代码、脚本和数据处理逻辑。
部分词库输入可能来自外部开源项目或公开资源转换而成。

## Fcitx 相关来源

- 本仓库中的部分词库转换流程可以导入来自 Fcitx 相关项目的源表。
- 这些源表的原始版权和许可归其原作者与上游项目所有。
- 本仓库不会对第三方源表重新授权。

当前仓库里已经保存了以下 Fcitx 格式源表快照：

- `third_party/fcitx/zhengma.txt`
- `third_party/fcitx/zhengma-large.txt`
- `third_party/fcitx/zhengma-pinyin.txt`

这些文件可以通过文件头识别为 Fcitx 码表，例如：

- `;fcitx Version 0x03 Table file`
- `;fcitx 版本 0x03 码表文件`

截至目前，本仓库已经能明确记录其来源类型和来源家族，但尚未在这些快照旁边保存独立的上游许可文本。
因此，在正式公开发布到 GitHub 前，维护者应补充核验并记录：

1. 每个文件对应的准确上游仓库或发布包来源。
2. 对应的上游许可标识与许可全文。
3. 上游是否要求额外署名、保留说明或限制再分发。

如果你分发基于 Fcitx 相关来源生成的词库文件，必须：

1. 保留上游项目的原始署名与许可说明。
2. 遵守上游源码或数据的许可条款。
3. 在发布包中同时提供来源链接和许可文本。
4. 如果无法核实上游许可，则不要再分发该源表或其派生结果。
