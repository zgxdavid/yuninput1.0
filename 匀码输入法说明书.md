# 匀码输入法说明书

更新日期：2026-05-16

当前建议测试版本：1.3.9

## 1. 快速开始

- 安装完成后，在 Windows 输入法列表中启用匀码输入法。
- 常用打开方式：按 Ctrl+F2 打开配置程序。
- 当前仓库根目录提供最新长测安装包：`Yuninput1.3.9.msi`。

## 2. 常用快捷键

- 左 Shift 轻触：中英文切换（从中文切到英文时会同时回到半角标点/半角字符）。
- F10：全角/半角字符切换。
- Ctrl+F2：打开配置程序。
- F2 或 APPS：打开状态菜单（F2 在菜单不可用时会回退打开配置程序）。
- Ctrl+Shift+F3 / Ctrl+Shift+F4：切换码表模式。
- Ctrl+Shift+M：把当前候选晋升为手工用户词条。
- Ctrl+1 到 Ctrl+9：置顶对应候选。
- Ctrl+Delete：屏蔽当前候选。
- [ 或 PgUp：上一页候选。
- ] 或 PgDn：下一页候选。
- , 或 .：翻页（等价于上一页/下一页）。

## 2.1 编码输入补充

- 0 通配符：仅在四码输入时生效，且只允许出现在第 2 位或第 4 位。
- 支持样例：a0cd、abc0、a0c0。
- 拼音查郑码：切到 zhengma-large-pinyin 模式后，按拼音检索单字，候选右侧显示对应郑码提示。

## 3. 配置程序说明

### 常规设置

- 可调整候选页大小、自动顶屏最短码长、回车策略、上下文学习开关和词库模式。
- “打开说明书”会直接用记事本打开本文件。

### 管理用户词典

- 这里统一管理手工用户词条和自动造词。
- 支持按关键字过滤、区分手工词条和自动造词、修改编码和词条内容。
- 修改完成后请点击“保存”，程序会分别写回用户词典和自动造词文件。

### 造词审阅日志

- 这里主要查看自动造词快照和手工造词日志。
- 若需要真正编辑词条，请回到“管理用户词典”。

### 上下文学习

- 可查看、导出、裁剪、拉黑或重建上下文学习数据。

## 4. 数据文件位置

- 用户数据目录：%APPDATA%\yuninput
- 常见文件：
  - yuninput_user.dict：手工用户词条。
  - auto_phrase_runtime.dict：运行时自动造词快照。
  - user_freq.txt：用户学习频次。
  - context_assoc.txt：上下文学习数据。
  - manual_phrase_review.txt：手工造词审阅日志。

## 5. 说明

- 若修改配置后没有立即生效，可先保存配置，再切换一次输入法或重新激活当前应用中的匀码输入法。
- 如果需要手工检查数据文件，建议先关闭配置程序，避免同时编辑同一文件。

## 6. 1.3.9 本轮更新

- 修复配置程序“造词审阅日志”来源筛选与主程序写入标记不一致的问题：`仅手工造词`/`仅自动造词`筛选现在可正确命中。
- 配置程序将“打开造词审阅日志”按钮调整为“手工造词日志”，并默认打开手工视图，降低“功能消失”的误判。
- 修复候选快速路径下会话自动造词未合并的问题，避免昨天输入过的潜在词在今天检索时消失。
- 会话自动造词状态改为运行时初始化阶段优先加载，避免因延迟预热被持续输入推迟导致历史词未生效。
- MSI 打包脚本默认升级到 1.3.9，并在配置程序源码更新时自动重编配置程序，避免把旧版 `yuninput_config.exe` 打进新包。
